# InnovAero Academic LaTeX Report

This project is set up as a polished academic report template for the InnovAero competition.

## Structure

- `main.tex`: report class, title page, metadata, and bibliography setup
- `chapters/related_work.tex`: structured related-work chapter with competition context
- `references.bib`: bibliography database

## Build

Compile with pdfLaTeX + BibTeX.

## Next steps

1. Replace `Team Name` and `University Affiliation` in `main.tex`.
2. Add your competition-specific figures, assumptions, and measurements.
3. Extend the chapter with a concept section, evaluation section, or simulation results as needed.

## Language

The report content is in English so it can be handed over in a professional technical format.
